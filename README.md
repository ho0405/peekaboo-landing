# Peekaboo Landing Page

Peekaboo 앱 소개 및 다운로드 랜딩 페이지입니다.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## 개발

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## Vercel 배포

1. [vercel.com](https://vercel.com)에 로그인
2. **Add New Project** → 이 저장소 import
3. **Deploy** 클릭

또는 Vercel CLI:

```bash
npx vercel
```
